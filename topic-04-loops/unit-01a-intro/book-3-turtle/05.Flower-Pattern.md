# 5. Flower Pattern

Loops can repeat entire shapes.

In this activity, you will draw a flower by drawing several circles and turning slightly each time.

## Flower Code

~~~python
import turtle

screen = turtle.Screen()
screen.bgcolor("white")

flower = turtle.Turtle()
flower.speed(0)
flower.color("purple")

for petal in range(18):
    flower.circle(80)
    flower.left(20)

turtle.done()
~~~

## What is happening?

- The turtle draws a circle.
- It turns left by 20 degrees.
- The loop repeats 18 times.

Because:

~~~text
18 x 20 = 360
~~~

the turtle eventually completes a full rotation.
