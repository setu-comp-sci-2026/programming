# 1. Meet the Turtle

Python has a built-in graphics module called `turtle`.

A turtle is like a pen on the screen. When the turtle moves, it can draw a line.

## Step 1

Create a file called `01_first_turtle.py`.

Enter the following code:

~~~python
import turtle

screen = turtle.Screen()
screen.title("My First Turtle Drawing")

tom = turtle.Turtle()

tom.forward(100)
tom.right(90)
tom.forward(100)

turtle.done()
~~~

## What is happening?

- `import turtle` allows us to use turtle graphics.
- `turtle.Screen()` creates the drawing window.
- `turtle.Turtle()` creates a turtle object.
- `forward(100)` moves the turtle forward 100 pixels.
- `right(90)` turns the turtle 90 degrees.
- `turtle.done()` keeps the window open.
