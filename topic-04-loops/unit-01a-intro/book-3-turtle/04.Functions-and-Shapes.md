# 4. Using Functions to Draw Shapes

You already know how to write functions.

Functions are very useful with turtle because they allow us to reuse drawing code.

## Step 1

Create a file called `04_functions_and_shapes.py`.

~~~python
import turtle

def draw_square(t, size):
    for side in range(4):
        t.forward(size)
        t.right(90)

screen = turtle.Screen()
artist = turtle.Turtle()

draw_square(artist, 100)
draw_square(artist, 50)

turtle.done()
~~~

## What is happening?

The function:

~~~python
def draw_square(t, size):
~~~

takes two values:

- `t` is the turtle
- `size` is the side length

## Improve It

Move the turtle before drawing the second square.

~~~python
artist.penup()
artist.forward(150)
artist.pendown()
draw_square(artist, 50)
~~~
