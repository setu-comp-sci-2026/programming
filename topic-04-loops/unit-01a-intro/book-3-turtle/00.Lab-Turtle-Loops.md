# Fun Lab: Drawing with Turtle and Loops

## Theme

In this lab you will use Python's `turtle` module to draw shapes, flowers, spirals, patterns and simple structures.

Turtle is a beginner-friendly graphics tool in Python. You control a small arrow, called a turtle, by telling it to move forward, turn left, turn right, change colour, lift the pen, put the pen down, and draw.

This lab focuses on using loops to avoid repeating the same drawing commands over and over.

## Learning Outcomes

By the end of this lab you should be able to:

- explain what turtle graphics are
- create a turtle drawing window
- move and turn a turtle
- use loops to draw repeated shapes
- use functions to organise drawing code
- create patterns using repetition
- combine loops, selection and functions in a creative program

## Before You Start

Create a new Python project called **TurtleLoopsLab**.

Each activity can be created in a separate `.py` file.

Suggested filenames:

- `01_first_turtle.py`
- `02_square.py`
- `03_shapes.py`
- `04_flower.py`
- `05_spiral.py`
- `06_structure.py`
- `07_final_design.py`
