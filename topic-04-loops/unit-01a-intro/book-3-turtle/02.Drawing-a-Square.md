# 2. Drawing a Square

A square has:

- 4 sides
- 4 right turns
- each turn is 90 degrees

Without a loop we could write:

~~~python
import turtle

screen = turtle.Screen()
shelly = turtle.Turtle()

shelly.forward(100)
shelly.right(90)
shelly.forward(100)
shelly.right(90)
shelly.forward(100)
shelly.right(90)
shelly.forward(100)
shelly.right(90)

turtle.done()
~~~

This works, but it repeats the same instructions.

## Better Version Using a Loop

~~~python
import turtle

screen = turtle.Screen()
shelly = turtle.Turtle()

for side in range(4):
    shelly.forward(100)
    shelly.right(90)

turtle.done()
~~~

## Questions
