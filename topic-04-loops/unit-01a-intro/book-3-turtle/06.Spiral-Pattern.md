# 6. Spiral Pattern

A spiral is created by moving forward a little more each time.

## Spiral Code

~~~python
import turtle

screen = turtle.Screen()
spiral = turtle.Turtle()
spiral.speed(0)

length = 5

for line in range(60):
    spiral.forward(length)
    spiral.right(90)
    length = length + 5

turtle.done()
~~~

## What is happening?

The variable `length` changes each time through the loop.

This means the line gets longer and longer.

## This Uses an Accumulator Pattern

~~~python
length = length + 5
~~~

or

~~~python
length += 5
~~~

## Try It

Change the turn angle:

~~~python
spiral.right(90)
spiral.right(91)
spiral.right(120)
spiral.right(60)
~~~
