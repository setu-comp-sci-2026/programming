# 7. Drawing a Simple Structure

In this activity you will draw a small house using functions.

You will need:

- a square for the house
- a triangle for the roof
- a rectangle for the door
- small squares for windows

## Starter Code

~~~python
import turtle

def draw_square(t, size):
    for side in range(4):
        t.forward(size)
        t.right(90)

def draw_triangle(t, size):
    for side in range(3):
        t.forward(size)
        t.left(120)

screen = turtle.Screen()
builder = turtle.Turtle()
builder.speed(3)

# Draw house body
draw_square(builder, 150)

# Move to roof position
builder.left(90)
builder.forward(150)
builder.right(90)

# Draw roof
draw_triangle(builder, 150)

turtle.done()
~~~
