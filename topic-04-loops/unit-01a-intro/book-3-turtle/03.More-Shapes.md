# 3. Drawing More Shapes

Many shapes can be drawn using the same pattern:

1. Move forward.
2. Turn.
3. Repeat.

## Triangle

A triangle has 3 sides.

~~~python
import turtle

screen = turtle.Screen()
t = turtle.Turtle()

for side in range(3):
    t.forward(120)
    t.left(120)

turtle.done()
~~~

## Pentagon

A pentagon has 5 sides.

~~~python
import turtle

screen = turtle.Screen()
t = turtle.Turtle()

for side in range(5):
    t.forward(100)
    t.left(72)

turtle.done()
~~~

## Why 72?

A full turn is 360 degrees.

For a regular pentagon:

~~~text
360 / 5 = 72
~~~

## General Rule

To draw a regular shape:

~~~python
angle = 360 / number_of_sides
~~~
