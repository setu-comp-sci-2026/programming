# # Getting to know the 'while' loop

while loops · validation · sentinel values · menu loops · functions

By the end of this lab, you should be able to:

- Explain when a `while` loop is useful.
- Write a `while` loop that repeats while a condition is true.
- Use a counter variable to control repetition.
- Use a `while` loop to validate user input.
- Use a sentinel value to decide when to stop.
- Use a menu loop to keep a program running until the user chooses to exit.
- Organise repeated tasks using functions.

## Before you begin

You should already be comfortable with:

- variables
- `input()` and `print()`
- `if`, `elif`, `else`
- Boolean operators: `and`, `or`, `not`
- writing and calling functions
- running Python programs in Visual Studio Code

## Suggested project setup

1. Open Visual Studio Code.
2. Create a new folder called **WhileLoopsLab**.
3. Create a separate Python file for each activity.
4. Run each file before moving on.

Suggested files:

- `01_counting.py`
- `02_validation.py`
- `03_password.py`
- `04_sentinel.py`
- `05_menu.py`
- `06_functions_and_loops.py`
