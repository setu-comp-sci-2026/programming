# Challenges

## Challenge 1 - Guess the number

Create a program where the user has to guess a secret number.

Requirements:

- Store a secret number in a variable.
- Ask the user to guess the number.
- Keep asking while the guess is incorrect.
- Tell the user whether the guess is too high or too low.
- Count the number of guesses.

Example:

~~~
Guess the number: 5
Too low
Guess the number: 9
Too high
Guess the number: 7
Correct
Number of guesses: 3
~~~

## Challenge 2 - Improved password program

Write a password program with the following rules:

- The correct password is `"python123"`.
- The user has 3 attempts.
- If the user enters the correct password, print `"Access granted"`.
- If the user uses all attempts, print `"Account locked"`.
- Print how many attempts are remaining after each incorrect attempt.

## Challenge 3 - Student mark system

Create a menu-driven program with these options:

~~~
1. Enter a valid mark
2. Show grade
3. Exit
~~~

Rules:

- A mark must be between 0 and 100.
- Use a function called `get_valid_mark()`.
- Use a function called `get_grade(mark)`.
- The grade should be:
  - 70 or above: Distinction
  - 60 to 69: Merit
  - 40 to 59: Pass
  - Below 40: Fail

## Challenge 4 - Simple bank menu

Create a bank account program.

The program should have a balance that starts at 0.

Menu:

~~~
1. Deposit
2. Withdraw
3. View balance
4. Exit
~~~

Rules:

- The menu should repeat until the user chooses Exit.
- Deposit must be positive.
- Withdrawal must be positive.
- The user cannot withdraw more than the balance.
- Use functions where appropriate.
