# Exercises

Complete these exercises after working through the examples.

## Exercise 1 - Repeat a message

Write a program that asks the user for their name and a number.

The program should print:

~~~
Hello name
~~~

that number of times.

Example:

~~~
Enter name: Sam
How many times? 3
Hello Sam
Hello Sam
Hello Sam
~~~

## Exercise 2 - Positive number validator

Write a function called `get_positive_number()`.

The function should:

- ask the user to enter a positive number
- keep asking while the number is 0 or less
- return the positive number

Call the function and print the returned value.

## Exercise 3 - Mark validator

Write a function called `get_valid_mark()`.

The mark must be between 0 and 100.

Then write a program that:

1. Gets a valid mark.
2. Prints `"Pass"` if the mark is 40 or above.
3. Prints `"Fail"` otherwise.

## Exercise 4 - Count spaces

Write a program that asks the user for a sentence and counts how many spaces it contains.

Example:

~~~
Enter a sentence: Python is fun
Number of spaces: 2
~~~

Hint:

You can use an index and a `while` loop:

~~~python
index = 0

while index < len(sentence):
    # check sentence[index]
    index = index + 1
~~~

## Exercise 5 - Total until stop

Write a program that asks the user to enter numbers.

The user should enter `0` to stop.

The program should print the total of the numbers entered.

## Exercise 6 - Menu program

Create a menu-driven program with the following options:

~~~
1. Print welcome message
2. Double a number
3. Check if a number is even or odd
4. Exit
~~~

Use functions for each option.
