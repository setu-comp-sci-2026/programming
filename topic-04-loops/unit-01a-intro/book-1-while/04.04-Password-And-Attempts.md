# Password loops and attempts

A `while` loop is useful when you do not know how many times the user will need to try.

## Example 1 - Keep asking for a password

Create a file called **04_password.py**.

~~~python
password = input("Enter password: ")

while password != "python":
    print("Incorrect password")
    password = input("Enter password: ")

print("Access granted")
~~~

Run the program.

## Example 2 - Count the number of attempts

~~~python
password = input("Enter password: ")
attempts = 1

while password != "python":
    print("Incorrect password")
    password = input("Enter password: ")
    attempts = attempts + 1

print("Access granted")
print("Number of attempts:", attempts)
~~~

## Example 3 - Use `and`

Limit the user to 3 attempts.

~~~python
password = input("Enter password: ")
attempts = 1

while password != "python" and attempts < 3:
    print("Incorrect password")
    password = input("Enter password: ")
    attempts = attempts + 1

if password == "python":
    print("Access granted")
else:
    print("Too many attempts")
~~~
