# Validation loops

A validation loop is used when the user must enter valid data.

You already know how to use selection to check a value once:

~~~python
mark = int(input("Enter mark: "))

if mark < 0 or mark > 100:
    print("Invalid mark")
~~~

But this only checks the mark once.

A `while` loop lets us keep asking until the value is valid.

## Example 1 - Validate a mark

Create a file called **03_validation.py**.

~~~python
mark = int(input("Enter a mark between 0 and 100: "))

while mark < 0 or mark > 100:
    print("Invalid mark")
    mark = int(input("Enter a mark between 0 and 100: "))

print("Valid mark entered:", mark)
~~~

Run the program several times.

Try entering:

- `75`
- `-4`
- `105`
- `0`
- `100`

## Why do we use `or`?

The mark is invalid if:

- it is less than 0

or

- it is greater than 100

~~~python
mark < 0 or mark > 100
~~~

## Example 2 - Validate an age

Write a program that asks the user for their age.

The age must be between 1 and 120.

Starter code:

~~~python
age = int(input("Enter your age: "))

while age < 1 or age > 120:
    print("Invalid age")
    age = int(input("Enter your age: "))

print("Age accepted:", age)
~~~

## Example 3 - Use a function

Create a function called `get_valid_mark()`.

~~~python
def get_valid_mark():
    mark = int(input("Enter a mark between 0 and 100: "))

    while mark < 0 or mark > 100:
        print("Invalid mark")
        mark = int(input("Enter a mark between 0 and 100: "))

    return mark


student_mark = get_valid_mark()
print("The valid mark is:", student_mark)
~~~
