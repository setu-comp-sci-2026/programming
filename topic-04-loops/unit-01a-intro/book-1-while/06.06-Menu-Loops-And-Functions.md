# Menu loops and functions

Many programs repeat a menu until the user chooses to exit.

In this step, you will combine:

- `while` loops
- selection
- functions

## Example 1 - Basic menu loop

Create a file called **06_menu.py**.

~~~python
choice = ""

while choice != "4":
    print("1. Say hello")
    print("2. Add two numbers")
    print("3. Print a message")
    print("4. Exit")

    choice = input("Enter your choice: ")

    if choice == "1":
        print("Hello")
    elif choice == "2":
        num1 = int(input("Enter first number: "))
        num2 = int(input("Enter second number: "))
        print("Total:", num1 + num2)
    elif choice == "3":
        print("Python is fun")
    elif choice == "4":
        print("Goodbye")
    else:
        print("Invalid choice")
~~~

Run the program and test each menu option.

## Example 2 - Move repeated tasks into functions

Create a new file called **06_functions_and_loops.py**.

~~~python
def print_menu():
    print("1. Say hello")
    print("2. Add two numbers")
    print("3. Print a message")
    print("4. Exit")


def say_hello():
    print("Hello")


def add_two_numbers():
    num1 = int(input("Enter first number: "))
    num2 = int(input("Enter second number: "))
    print("Total:", num1 + num2)


def print_message():
    print("Python is fun")


choice = ""

while choice != "4":
    print_menu()
    choice = input("Enter your choice: ")

    if choice == "1":
        say_hello()
    elif choice == "2":
        add_two_numbers()
    elif choice == "3":
        print_message()
    elif choice == "4":
        print("Goodbye")
    else:
        print("Invalid choice")
~~~

## Why use functions?

Functions make the program:

- easier to read
- easier to test
- easier to change
- less repetitive
