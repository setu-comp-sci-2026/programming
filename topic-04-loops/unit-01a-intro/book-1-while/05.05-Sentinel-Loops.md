# Sentinel loops

A sentinel is a special value that means **stop**.

For example:

- entering `-1` to stop entering numbers
- entering `"quit"` to stop a program
- choosing option `0` to exit a menu

## Example 1 - Enter numbers until -1

Create a file called **05_sentinel.py**.

~~~python
number = int(input("Enter a number (-1 to stop): "))

while number != -1:
    print("You entered:", number)
    number = int(input("Enter a number (-1 to stop): "))

print("Finished")
~~~

Run the program.

## Example 2 - Add the numbers

~~~python
total = 0

number = int(input("Enter a number (-1 to stop): "))

while number != -1:
    total = total + number
    number = int(input("Enter a number (-1 to stop): "))

print("Total:", total)
~~~

## Example 3 - Count how many numbers were entered

~~~python
total = 0
count = 0

number = int(input("Enter a number (-1 to stop): "))

while number != -1:
    total = total + number
    count = count + 1
    number = int(input("Enter a number (-1 to stop): "))

print("Total:", total)
print("Numbers entered:", count)
~~~

## Example 4 - Avoid division by zero

Calculate the average.

~~~python
total = 0
count = 0

number = int(input("Enter a number (-1 to stop): "))

while number != -1:
    total = total + number
    count = count + 1
    number = int(input("Enter a number (-1 to stop): "))

if count > 0:
    average = total / count
    print("Average:", average)
else:
    print("No numbers were entered")
~~~
