# Understanding while loops

A `while` loop repeats a block of code **while a condition is true**.

You already know how to make a decision once:

~~~python
age = int(input("Enter your age: "))

if age >= 18:
    print("Adult")
else:
    print("Minor")
~~~

A loop lets us repeat a decision.

## Basic structure

~~~python
while condition:
    statements
~~~

The statements inside the loop must be indented.

## Example 1 - Print a message 5 times

Create a file called **01_counting.py**.

Enter the following code:

~~~python
count = 1

while count <= 5:
    print("Welcome to Python")
    count = count + 1
~~~

Run your code.

## What is happening?

The variable `count` controls the loop.

| Part | Code |
| --- | --- |
| Starting value | `count = 1` |
| Condition | `count <= 5` |
| Repeated action | `print("Welcome to Python")` |
| Update | `count = count + 1` |
