# Challenges

## Challenge 1 – Password Strength Checker

Ask the user for a password.

Use a loop to count:

- uppercase letters
- lowercase letters
- digits

Display the totals.

## Challenge 2 – Word Counter

Ask the user for a sentence.

Count the number of words.

Hint:

Words = Spaces + 1

## Challenge 3 – Times Table Generator

Ask the user for a number.

Display its multiplication table.

## Challenge 4 – Character Frequency

Ask the user for a sentence.

Ask for a character.

Count how many times the character occurs.
