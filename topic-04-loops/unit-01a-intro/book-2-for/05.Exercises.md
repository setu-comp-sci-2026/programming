# Exercises

## Exercise 1 – Counting

Write a program that displays:

1
2
3
...
20

## Exercise 2 – Even Numbers

Display all even numbers between 2 and 50.

## Exercise 3 – Multiples of Three

Display all multiples of three between 1 and 30.

## Exercise 4 – Name Printer

Ask the user for their name.

Print each character on a separate line.

## Exercise 5 – Count Spaces

Ask the user for a sentence.

Count how many spaces it contains.

## Exercise 6 – Count Vowels

Ask the user for a sentence.

Count how many vowels it contains.
