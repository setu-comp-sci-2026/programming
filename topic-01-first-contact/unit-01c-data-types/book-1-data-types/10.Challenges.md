#Challenges

In this step, we will create Python programs that use variables and arithmetic operators to calculate and display results.

For each challenge listed below, create a new Python file in VS Code.


##Challenge 1

Create a new Python file and in it, create two new variables:
- **xCoord** which is set to 300
- **yCoord** which is also set to 300

Then write code that uses **all four arithmetic operators** (+, -, *, /) on these two variables and prints the results in a clear format.  For example:

~~~python
print("xCoord + yCoord =", xCoord + yCoord)
print("xCoord - yCoord =", xCoord - yCoord)
print("xCoord * yCoord =", xCoord * yCoord)
print("xCoord / yCoord =", xCoord / yCoord)
~~~

Run the program and check the output.  Then try changing the values of xCoord and yCoord and observe how all the results update automatically.

Save your work.


##Challenge 2

In this challenge, we will use the solution from Challenge 1 above.  If you had difficulty with Challenge 1, you can find the solution in the solutions tab.

Create a new Python file and copy your Challenge 1 solution into it.

This time, instead of using *xCoord* and *yCoord* directly, divide xCoord by 2 and yCoord by 4 when printing the results.  Store these derived values in new variables:

~~~python
newX = xCoord / 2
newY = yCoord / 4
~~~

Then print both the original and derived values so you can compare them:

~~~python
print("Original xCoord:", xCoord, "  New xCoord:", newX)
print("Original yCoord:", yCoord, "  New yCoord:", newY)
~~~

Save your work.


##Challenge 3

In this challenge, we will extend Challenge 1 to work with two sets of coordinates.

Create a new Python file and in it, create two new variables:
- **xCoord** which is set to 300
- **yCoord** which is also set to 300

Calculate and print two sets of results:
- A first set using **xCoord** and **yCoord** directly
- A second set using **xCoord - 100** and **yCoord - 100**

Label each set clearly in the output so you can see both sets of values side by side.

Run the program and observe how the two sets of values differ.

Save your work.
